## Installation 
`yarn global add knit` _or_ `npm install -g knit`


## Basic usage 
* create a [new project](https://github.com/new) with same name as your folder
* go to your local project folder 
* simply run `knit`. 
  
## First time configuration
`knit --user  <your-user-name>`
